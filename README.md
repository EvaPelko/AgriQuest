# Welcome to AgriQuest!

![Placeholder](https://placehold.co/600x200/2B2F36/35D7BB?text=Your+Image+Here)


A 2D pixel game about terraforming planets in a galaxy far far away...

---

## Game Engine

AgriQuest was made using Godot version x.x
Godot is a free open source game engine capable of incredible things.

## Features

In AgriQuest you are able to:

- Explore the planet system called Orchid
- Terraform planets 
- Save your species!
